# Redstone Master

Клиентский Fabric-мод для Minecraft **1.21.11** с обучающими материалами по редстоун-механикам.

Проект является **выпускной квалификационной работой (ВКР)**.  
Автор: **Смирнов Григорий Витальевич**.

## О проекте

Мод открывается поверх игры и не требует серверной части (`environment: client`). Сейчас реализованы интерфейс, настройки и локализация; учебный контент на вкладках **Обучение** и **Профиль** находится в разработке.

## Возможности

- Окно мода по клавише **]** (правая квадратная скобка); повторное нажатие закрывает окно (настраивается)
- Центральная панель с масштабом **60–100%** экрана (по умолчанию **80%**); по краям виден мир или фон главного меню
- Вкладки:
  - **Главное меню** — приветствие и описание проекта
  - **Обучение** — заглушка (планируются уроки по редстоуну)
  - **Настройки** — размер окна, пауза, контраст рамок, язык, сохранение сессии
  - **Профиль** — заглушка (планируется информация об авторе и ВКР)
- Локализация интерфейса мода: **русский** и **английский** (автоподбор по языку Minecraft или ручной выбор)
- Конфигурация в `config/redstone-master.json`

## Требования

| Компонент | Версия |
|-----------|--------|
| Minecraft | 1.21.11 |
| Fabric Loader | ≥ 0.19.2 |
| Fabric API | 0.141.4+1.21.11 |
| JDK | 21 |

## Сборка

```bash
./gradlew build
```

Готовый JAR: `build/libs/redstone-master-1.0.0.jar`

На Windows можно использовать `gradlew.bat`:

```bat
gradlew.bat build
```

## Запуск в среде разработки

```bash
./gradlew runClient
```

или `run.bat` / `gradlew.bat runClient` в Windows.

## Управление

| Действие | Клавиша / элемент |
|----------|-------------------|
| Открыть окно мода | `]` |
| Закрыть окно | Кнопка **X**, `Esc` |
| Закрыть повторным нажатием `]` | Включается в настройках мода («Закрывать мод по повторному нажатию») |

Клавишу открытия можно переназначить: **Настройки → Управление → Redstone Master**.

## Настройки мода

Доступны на вкладке **Настройки** внутри окна мода:

- **Размер окна мода** — 60%, 70%, 80%, 90%, 100%
- **Пауза при открытии** — в одиночной игре ставит мир на паузу
- **Высокий контраст рамок**
- **Автоподбор языка** / **Выбор языка** (ru / en)
- **Сохранять вкладку и позицию** — последняя вкладка и прокрутка списка настроек
- **Закрывать мод по повторному нажатию**
- Поиск по настройкам, сброс одной настройки или всех сразу
